# Lista de Compras

## Kit principal (cubre P0-P4)

### Opción recomendada: ELEGOO Super Starter Kit

**Precio:** ~35€
**Amazon:** Buscar "ELEGOO UNO R3 Super Starter Kit"

Incluye:
- Arduino UNO R3 (compatible)
- Cable USB
- Breadboard 830 puntos
- Cables jumper (M-M, M-F)
- LEDs (rojo, amarillo, verde, azul)
- Resistencias variadas (220Ω, 1kΩ, 10kΩ)
- Potenciómetro
- Sensor ultrasonido HC-SR04
- Servo SG90
- Buzzer activo y pasivo
- Fotorresistencias (LDR)
- Pulsadores
- Y más cosas que no usaremos aún

## LEGO recomendado

### Para P1-P4 (LEGO normal)
Cualquier LEGO que tengáis sirve. El niño construye lo que quiera.

### Para P5 (Robot)
**LEGO Technic** facilita mucho el montaje de motores:
- Vigas con agujeros (beams)
- Conectores (pins)
- Ejes

**Opciones:**
1. Set Technic barato (~15-20€) y desmontar
2. Lote de piezas Technic sueltas (Wallapop, BrickLink)
3. Combinar LEGO normal + adaptadores impresos en 3D
